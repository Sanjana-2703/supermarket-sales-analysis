
# Supermarket Sales Analysis 🛒

## Overview
Analysis of 1000+ supermarket transactions across 3 cities using Python.

## Tools Used
- Python, Pandas, Matplotlib, Seaborn

## Business Questions Answered
1. Which city has the highest sales?
2. Which product line performs best?
3. How do customers prefer to pay?
4. How satisfied are customers?
5. Which month had the highest sales?

## Key Findings
- Naypyitaw had the highest total sales
- Food & Beverages was the best selling product line
- Ewallet was the most popular payment method
- Average customer rating: 6.97 out of 10
- January had the highest sales

## Dataset
1000 transactions | 17 columns | 3 cities
